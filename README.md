# Numpy Command Line Interface (CLI)

This project provides a command-line interface (CLI) for the python numpy package.

## Installation

To install and validate `numpy-cli`:

~~~
sh# git clone https://github.com/dchassin/numpy-cli
sh# cd numpy-cli
sh# make install
sh# make validate
~~~
